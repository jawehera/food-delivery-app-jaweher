# Food Delivery App

## Description

Food Delivery App is a mobile application developed with React Native and Expo SDK 56. The application allows users to browse restaurants, search by cuisine, view restaurant details and menus, manage their cart, place orders, and customize their profile with personal information and a profile picture.

---

## Features

* Browse available restaurants
* Search restaurants by name
* Filter restaurants by category
* View restaurant details and menus
* Add and remove items from the cart
* View order summary
* Create and edit a user profile
* Upload and update a profile picture
* Persistent local storage using AsyncStorage
* Responsive and modern mobile UI

---

## Tech Stack

* React Native
* Expo SDK 56
* Expo Router
* TypeScript
* AsyncStorage

## Setup Instructions

### 1. Clone the repository

```bash
git clone <repository-url>
cd foodproject
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the application

```bash
npx expo start
```

### 4. Run on a device

* Press **a** to launch Android Emulator
* Press **i** to launch iOS Simulator
* Scan the QR Code using Expo Go on a physical device
